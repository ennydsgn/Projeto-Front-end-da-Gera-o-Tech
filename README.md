## Executando

##### Clonar o repositório
> `git clone https://github.com/letticiamoura/ecommerce-front.git`
> 
##### Entrar na pasta
> `cd ecommerce-front`

##### Instalar dependências
> `npm i`

##### Inicializando projeto
> `npm dev run`
